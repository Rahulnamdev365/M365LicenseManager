# Contributing

Thank you for your interest in contributing to M365LicenseManager.

## Reporting Issues

If you encounter a bug, please open a GitHub Issue and include:

- PowerShell version
- Module version
- Error message
- Steps to reproduce

## Feature Requests

Feature requests are welcome. Please create an issue describing the proposed functionality and its use case.

## Pull Requests

Before submitting a pull request:

- Follow PowerShell best practices.
- Update comment-based help if required.
- Keep functions small and reusable.
- Test your changes before submitting.
